- Para executar o programa é necessário que seja criado um arquivo TXT chamado "parametros_entrada.txt" e adicionado a pasta onde esta o arquivo executavel chamado "main.exe". Este arquivo deve seguir a seguinte estrutura:
		- Estrutura: 10,merge_sort
			- O numero "10" é o tamanho do array
			- A palavra "merge_sort" é o algoritmo que será executado. As opções de algoritmos são: 
				i) merge_sort
				ii) intro_sort
				ii) heap_sort

- O resultado da execução será em um arquivo csv chamdado "resultados.csv".
